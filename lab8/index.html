Lab 8 - Anonymous Functions & Callbacks

Challenge

The challenge of this lab was creating an anonymous function and a regular function. We first created our functions in browser console, then we it into a JavaScript file.

Problems

This Lab was more difficult than other ones, but I feel like I got a lot out of it considering we had to problem solve how to translate the direct console code into JavaScript.
